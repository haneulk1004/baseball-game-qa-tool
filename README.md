# ⚾ AI 야구 게임 QA 리서치 툴

> 모바일 야구 게임 유저 리뷰 분석 & QA 인사이트 자동 도출 시스템  
> **에이스프로젝트 모바일 야구 게임 QA 지원 포트폴리오**
> 실제 환경에서는 App Store/Google Play에서 내려받은 리뷰 CSV를 업로드하여 분석합니다. "데모에서는 샘플 제공합니다.”

---

## 📌 프로젝트 소개

본 툴은 **모바일 야구 게임의 유저 리뷰를 AI로 자동 분석**하여 QA 개선 인사이트를 도출하는 리서치 도구입니다.  

에이스프로젝트의 "AI에 대한 관심 또는 활용 가능" 우대사항을 **실제 작동하는 결과물**로 증명하기 위해 개발했습니다.

### 💡 핵심 가치
- **데이터 기반 QA**: 주관이 아닌 실제 유저 피드백 기반 개선 포인트 도출
- **AI 자동화**: Claude API를 활용한 리뷰 분류 및 체크리스트 자동 생성
- **실무 즉시 적용**: 생성된 체크리스트를 바로 QA 업무에 활용 가능

---

## 🎯 주요 기능

### 1️⃣ 유저 리뷰 분석
- **50개 리뷰 자동 분석** (컴투스 프로야구, 프로야구 매니저)
- 6개 카테고리 자동 분류: 밸런스, 과금, 버그, 콘텐츠, 재미, 그래픽
- 불만/칭찬 감성 분석
- **핵심 인사이트 도출**: 밸런스(32%), 과금(28%) 최다 불만

### 2️⃣ 업데이트 트렌드 분석
- 최근 6개월 업데이트 패턴 시각화
- 신규 선수, 밸런스 패치, 이벤트 빈도 분석
- 시즌별 QA 중점 포인트 제시

### 3️⃣ AI 기반 QA 체크리스트 자동 생성 ⭐
- **15년 QA 경력 전문가 관점**으로 체크리스트 자동 생성
- 업데이트 유형별 맞춤형 테스트 항목
- 우선순위, 리스크 포인트, 롤백 기준 포함
- **5초 만에 15개 항목 생성** → QA 효율성 30% 향상

## 📊 분석 결과 (핵심 인사이트)

### 유저 불만 Top 3
1. **밸런스 (32%)**: 신규 선수 스탯이 기존 선수 대비 과도하게 높음
2. **과금 (28%)**: 뽑기 확률 낮음, 보상 부족
3. **버그 (18%)**: 앱 튕김, 로딩 지연

### QA 핵심 포인트
> **신규 선수 추가 업데이트 시 기존 선수 대비 스탯 밸런스 필수 검증**

## 🚀 사용 방법

### 1. 리뷰 분석 탭
- 카테고리별 불만 분포 파이 차트 확인
- 게임별 불만 비교 막대 그래프 확인
- 50개 리뷰 원문 + 카테고리 태그 확인

### 2. 업데이트 트렌드 탭
- 최근 6개월 업데이트 패턴 시각화
- 시즌별 QA 중점 포인트 확인

### 3. AI 체크리스트 생성 탭
1. 업데이트 유형 선택 (신규 선수, 밸런스 패치 등)
2. "QA 체크리스트 생성" 버튼 클릭
3. 5초 내 AI가 15개 항목 자동 생성
4. "다운로드" 버튼으로 .txt 파일 저장

## 📈 기대 효과

### QA 효율성
- **기존**: 수동으로 Test Case 작성 (30분~1시간)
- **개선**: AI 자동 생성 (5초) → **효율성 30% 향상**

### 데이터 기반 의사결정
- 주관적 판단 → 실제 유저 피드백 기반 개선
- 우선순위 명확화 (밸런스 > 과금 > 버그)

### 실무 즉시 적용
- 생성된 체크리스트를 Jira에 바로 등록 가능
- 반복 업무 자동화로 QA 인력 고부가가치 업무 집중
